# Dragon News
Live link : https://dragonnews-12df3.web.app/catagories/1


**Project Overview:**  
Dragon News is a responsive news web application built with React, delivering  up-to-date news articles across various categories. I uses the Programming Hero News API to fetch categories, articles, and detailed news items. Users can browse news by category, view article details, and enjoy a clean, mobile-friendly UI. The app also includes Firebase authentication, enabling users to sign in and access a personalized experience.
